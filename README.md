# SMO_Randomizer_Tracker

A live tracker + OBS overlay for Super Mario Odyssey Ranomizer speedruns.

## Requirements
- Python 3.11+  
Download from: https://www.python.org/downloads/
✔ Make sure "Add Python to PATH" is checked

## Installation
1. Git clone
2. Install dependencies through a terminal in the project folder: `pip install -r requirements.txt`
3. Run the tracker: `python tracker.py`

> [!NOTE]
> The tracker will make a json file to save progress
